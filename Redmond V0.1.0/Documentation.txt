VERSION X.X.X; Specifies which interpreter to use. Must be the first line of code.
INT NAME,[VALUE]; Allocates mem for int NAME and sets it's value to VALUE. Defaults to 0.
SET @NAME, [VALUE]; Sets NAME's value to VALUE. Defaults to 0.
INC @NAME; Increments an integer.
DEC @NAME; Decrements an integer.
GREATER ARG1,ARG2,LINE; If arg1 > arg2, call JUMP LINE.
JUMP INT; Go to line INT.
EXIT; Exit the program.
PRINT INPUT1,INPUT2...; Prints all inputs. Inputs can be strings, variables, or numbers.
WINDOW; Open a window for drawing.
PIXEL X,Y,R,G,B; Draw a pixel at (X,Y) with color (R,G,B). Will only display on buffer swap if in manual buffer mode.
KEY $KEY,@OUT; Increments @OUT if key $KEY is pressed..
@VAR = variable.
$"STRING" = string.
